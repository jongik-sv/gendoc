## 콘텐츠 유형 분류 체계

| 대분류 | 세부 유형 |
|--------|----------|
| 텍스트 | 단순 텍스트, 불릿 리스트, 번호 리스트 |
| 박스형 | 2분할, 3분할, 4분할, 6분할, 다중분할 |
| 이미지 | 단일 이미지, 다중 이미지, 이미지+텍스트 |
| 차트 | 막대(세로/가로/누적), 선형, 파이, 도넛, 영역, 복합 |
| 다이어그램 | 프로세스, 사이클, 계층, 벤다이어그램, 마인드맵 |
| 타임라인 | 수평, 수직, 단계별 |
| 테이블 | 단순 테이블, 비교 테이블, 매트릭스 |
| 도형/아이콘 | 아이콘 기반, 도형 강조, 픽토그램 |
| 기타 | 타이틀, 목차, 섹션 구분, 조직도, 갠트차트 |

### 섹션 1: 타이틀 및 목차 (슬라이드 0-6)

| 슬라이드 | 콘텐츠 유형 | 상세 설명 |
|----------|------------|-----------|
| 0 | **타이틀 슬라이드** | 회사명(동국제강그룹) + 제목(PPT 템플릿 예시) + 부제목, 상단에 로고 배치, 중앙 정렬 레이아웃 |

## 레이아웃 패턴 요약

### 분할 레이아웃
- **2분할**: 좌우 또는 상하로 콘텐츠를 균등/비균등 분할
- **3분할**: 수평 3개 컬럼으로 균등 분할
- **4분할**: 2x2 그리드 또는 수평 4개 배열
- **6분할 이상**: 다중 항목을 그리드로 배치

### 차트 유형
- **막대 차트**: 세로형, 가로형, 그룹형, 누적형
- **선형 차트**: 단일 계열, 다중 계열, 영역형
- **원형 차트**: 파이, 도넛, 반원형
- **복합 차트**: 막대+선, 차트+테이블

### 다이어그램 유형
- **프로세스**: 선형(화살표), 순환형, 수직형
- **관계형**: 벤 다이어그램, 마인드맵
- **계층형**: 피라미드, 조직도
- **시간형**: 타임라인(수평/수직)

### 디자인 특징
- **색상 테마**: 네이비 블루(#1E3A5F) 기반, 회색/청록색 보조
- **헤더 바**: 상단에 파란색 헤더 바와 로고 배치
- **강조 요소**: 색상 박스, 아이콘, 큰 숫자로 핵심 정보 강조
- **그리드 시스템**: 일관된 여백과 정렬로 깔끔한 레이아웃
