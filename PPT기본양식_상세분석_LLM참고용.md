# PPT 기본양식 슬라이드별 상세 분석 보고서

**용도**: LLM이 PPT 생성 시 레이아웃 및 구조 참고용

## 사용 가이드

이 문서는 LLM이 PPT를 생성할 때 참고할 수 있도록 각 슬라이드의 구조, 도형 형태, 배치, 권장 용도를 상세히 기술합니다.

### 슬라이드 번호 체계
- 슬라이드 번호는 1부터 시작 (PowerPoint 페이지 번호와 동일)
- 각 슬라이드마다 레이아웃 구조, 시각적 형태, 권장 용도를 포함

### 슬라이드 1
**카테고리**: 표지/타이틀

**레이아웃 구조**:
- 전체 슬라이드가 진한 네이비 블루(#1E3A5F 계열) 배경
- 좌측 하단에서 우측 상단으로 흐르는 연한 회색의 대각선 X자 패턴(장식용 기하학적 요소)
- 좌측 상단에 회사 로고 배치

**시각적 형태**:
- 배경: 단색 진한 파란색
- 장식 요소: 얇은 선으로 구성된 X자 형태의 대각선 패턴이 슬라이드 중앙을 가로지름
- 로고: 좌측 상단 고정 위치

**권장 용도**:
- 프레젠테이션 표지
- 회사 소개 시작 페이지
- 세미나/발표 첫 페이지

### 슬라이드 7
**카테고리**: 목차

**레이아웃 구조**:

- 좌측: 현재 섹션 강조 박스 (파란색 배경)
- 우측: 목차 항목 리스트 (번호 + 항목명)

**시각적 형태**:
- 좌측에 강조된 현재 섹션 박스 (가로로 긴 직사각형, 파란색 채움)

- 각 항목은 가로로 긴 형태로 배치

**도형 상세**:
- 파란색 직사각형 박스 1개 (현재 섹션 표시용, 약 8cm x 1.5cm)
- 텍스트 항목들이 수직으로 나열

**권장 용도**:
- 프레젠테이션 목차 페이지
- 현재 위치 표시가 필요한 네비게이션
